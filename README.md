# 🎨 Hasta-Kala Shop (Micro-Sales Analytics)

**Hasta-Kala Shop** is a professional data-driven billing and analytics tool designed specifically for small-scale artisans and craftspeople. 

By transforming simple sales logging into actionable business intelligence, this app helps artisans eliminate "dead stock" and focus on producing their most profitable designs.

---

## 🚀 Key Features

### 📊 Real-Time Dashboard
*   **Total Revenue Tracking**: Monitor your financial growth at a glance.
*   **Revenue Analysis**: Interactive Bar Charts filterable by **Week, Month, and Year**.
*   **Best Seller Tracking**: Visual Pie Charts identifying top-performing colors and variants.
*   **Artisan Insights**: Automated intelligence that highlights your biggest revenue drivers and suggests what to make more of.

### 💸 Quick Bill System
*   **Rapid Sales Logging**: A tactile interface optimized for busy markets—log a sale in 3 clicks.
*   **Automatic Inventory Sync**: Every sale automatically reduces stock levels in real-time.

### 🚨 Proactive Stock Management
*   **Stock Alerts**: Automatic red banners warn you when an item drops to 2 or fewer units.
*   **Restock Page**: A dedicated tool to update your stock levels after you've produced new items.

### 📑 Professional Reporting
*   **Income Log**: A detailed history of every transaction.
*   **CSV Exports**: One-click downloads of your sales data for advanced analysis in Excel or Google Sheets.

## 🛠️ Tech Stack
*   **Python 3.x**
*   **Streamlit** (UI Framework)
*   **Pandas** (Data Handling)
*   **Plotly** (Interactive Analytics)

## 📦 Local Installation

If you want to run this app on your own computer:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/sainathsk48/hasta-kala-shop.git
    cd hasta-kala-shop
    ```

2.  **Install dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

3.  **Run the app**:
    ```bash
    streamlit run app.py
    ```

## ☁️ Deployment
This app is optimized for **Streamlit Community Cloud**. To deploy your own version:
1. Push this code to a GitHub repository.
2. Connect the repository to [share.streamlit.io](https://share.streamlit.io).
3. The app will automatically build and stay updated with every push.

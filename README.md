# 🎨 Hasta-Kala Shop (Streamlit Edition)

A micro-sales analytics tool for small artisans. This app allows craftspeople to log sales, track inventory, and visualize revenue trends.

## Features
- **Quick Bill**: Log sales in seconds.
- **Dashboard**: Real-time revenue and item performance analytics.
- **Stock Alerts**: Automatic warnings when inventory is low.
- **Income Log**: Filterable transaction history and CSV exports.

## Tech Stack
- Python
- Streamlit
- Pandas
- Plotly
